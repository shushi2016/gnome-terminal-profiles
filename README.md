<h1> Gnome Terminal Profiles</h1>

<h2> How to use </h2>
- Download the .dconf file
- rename it to *gnome-terminal-profiles.dconf*
- run this command
~~~~
  dconf load /org/gnome/terminal/legacy/profiles:/ < gnome-terminal-profiles.dconf
~~~~

- open a new terminal and enjoy!
